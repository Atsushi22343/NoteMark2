# NoteMark2

A minimal Note taking app with out-of-the-box markdown support.
This app is based on Gionatha's cool app. I am very grateful to him.

![Preview](https://github.com/Atsushi22343/NoteMark2/assets/demo.png)

## Project Setup

### Install

```bash
$ yarn
```

### Development

```bash
$ yarn dev
```

### Build

```bash
# For windows
$ yarn build:win

# For macOS
$ yarn build:mac

# For Linux
$ yarn build:linux
```
